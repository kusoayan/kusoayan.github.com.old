FuelPHP 安裝配置（一）

這系列記一下 FuelPHP 的安裝配置步驟，config、orm、auth、oil 等。

按照官方網站上的說明，可以藉由兩種方式來安裝：

  1. 直接透過指令安裝：

    $ curl get.fuelphp.com/oil | sh

    這種方式目前只在 *nix 系統（Linux、OS X、Unix、etc.）上可以使用，下了指令之後，他會自己把 oil 這個 shell-script 抓下來然後安裝到你的 /usr/bin 底下，也因此過程中他會需要你的密碼來授權。裝完之後可以看一下這個 shell-script 到底幹了什麼事情； “ bash oil

     #!/bin/bash

 if [ -f "./oil" ]; then
         php oil "$@"
 else

         if [ "$1" == "create" ]; then

                 if [ ! `which git` ]; then 
                     echo "For this installer to work you'll need to install Git."
                     echo '        http://git-scm.com/'
                 fi

                 git clone --recursive git://github.com/fuel/fuel.git "./$2"
                 php "./$2/oil" refine install
         else
                 echo 'This is not a valid Fuel installation so Oil is a bit lost.'
                 echo '        http://fuelphp.com/docs/installation/instructions.html'

         fi
 fi

    “

    其實就只是當你下指令 oil create 時去 clone 一份 FuelPHP Framework 的檔案回來，然後如果資料夾底下有 oil 這個檔案時，會自動幫你把 oil alias 到 php oil，也就是你可以直接用 oil 這個指令替代 php oil！

  2. 手動安裝

    1. 先下載 FeulPHP
    2. 把它解壓縮
    3. 把解壓縮後的資料夾移到你的 Server 上，結構大概如下：

    fuel/ app/ core/ packages/ public/ .htaccess assets/ index.php oil

    1. 設定資料夾的讀寫權限，可以手動或是執行底下的指令：

    “ $ php oil refine install

     Made writable: APPPATH/cache
 Made writable: APPPATH/logs
 Made writable: APPPATH/tmp

    “
