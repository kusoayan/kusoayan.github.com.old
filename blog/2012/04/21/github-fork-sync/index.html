Github Fork 後與原 Project 同步

在 github 上 fork 完一個 project 之後，是不會自動跟原本的 project 保持同步的，也就是說如果今天 fork 回來之後，原本的 project 又改了某些東西，那這些修改是不會出現在你的 fork 回來的 project 當中。 所以需要一些操作來讓其同步，首先 cd 進 git repo 的目錄：

cd some-project

然後將原本的 project 加入 remote 中

這樣就可以了！
