Octopress Content Manage Tool

Apr 17th, 2012

因為 Octopress 本身沒有草稿功能，參考了Octopress 程式碼與網站內容的管理作者提供的工具後，應該就能順利解決。

這邊紀錄一下在 archlinux 上的安裝步驟。

  1. 到 (https://github.com/hychen/octopress-admin) clone 檔案回來

  2. 安裝 ucltip，可以用 pip 安裝，不過要注意一下 pip 版本，我是用 pip2 install ucltip 來安裝。

  3. 把剛剛 clone 回來的 octopress-admin 丟到 ~/bin/ 底下，然後把 ~/bin/ 加到 PATH 中。

  4. 稍微改一下 octopress-admin 的內容，把最上面的

     #!/usr/bin/env python
    改成 
     #!/usr/bin/env python2

  5. 因為 archlinux 上好像沒有 sensible-editor 這個套件，所以我直接改成用 vim 來編輯，一樣修改 octopress-admin，我把第 17 行的

     ucltip.regcmds('rake', 'bundle', 'sensible-editor')
    改成了 
     ucltip.regcmds('rake', 'bundle', 'gvim')

    然後把檔案中的 sensible_editor 改成 givm，應該是有兩處要修改。

接下來就可以照著原作者的 usage 去做了！
